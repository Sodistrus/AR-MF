# AETH Spec Phase A

Phase A locks the language surface before compiler complexity.

Contents:
- `grammar.ebnf` — minimal grammar for canonical state/visual/rule blocks
- `semantics.md` — meaning and validation rules
- `standard-library.md` — canonical state, shape, palette, and rule names
- `versioning.md` — compatibility and change policy

Phase A also introduces golden fixtures that compile `.aeth` files into deterministic JSON payloads.
